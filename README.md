# Portfolio_Website

Simple static portfolio site scaffold with hero, bio, skills, and profile picture.

Files included:

- `index.html` — main page with hero, bio, and skills sections.
- `styles.css` — simple responsive styles.
- `assets/profile.svg` — small SVG profile placeholder.

To view locally:

```sh
# from workspace root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
